# Ray Tracer

A basic ray tracing engine written in C++. This project demonstrates foundational concepts of computer graphics, such as vector math, ray casting, and color computation.

## 🔧 Features

- Ray-object intersection
- Basic vector and color handling
- Simple scene rendering to the console

## 🛠️ Build Instructions

### Prerequisites

- C++17 compatible compiler (e.g., `g++`, `clang++`)
- [CMake](https://cmake.org/) 3.10 or higher (recommended)

### Build with CMake

```bash
git clone https://github.com/Yoshi-tech/Ray-tracer.git
cd Ray-tracer
mkdir build && cd build
cmake ..
make
